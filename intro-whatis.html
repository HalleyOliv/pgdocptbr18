1. O que é o PostgreSQL? #

O PostgreSQL é um sistema de gerenciamento de banco de dados objeto-relacional (ORDBMS/SGBDOR) baseado no POSTGRES, Version 4.2, desenvolvido no Departamento de Ciência da Computação da Universidade da Califórnia em Berkeley. O PostgreSQL foi pioneiro em muitos conceitos que só se tornaram disponíveis em alguns sistemas de banco de dados comerciais muito mais tarde.

O PostgreSQL é um descendente de código aberto desse código original de Berkeley. Oferece suporte a grande parte do padrão SQL e oferece muitos recursos modernos:

  • consultas complexas
  • chaves estrangeirass
  • gatilhos
  • visões atualizáveis
  • integridade transacional
  • controle de concorrência multiversão

Além disso, o PostgreSQL pode ser estendido pelo usuário de diversas maneiras, por exemplo, adicionando novos recursos.

  • tipos de dados
  • funções
  • operadores
  • funções de agregação
  • métodos de índice
  • linguagens procedurais

E, devido à sua licença liberal, o PostgreSQL pode ser usado, modificado e distribuído gratuitamente por qualquer pessoa para qualquer finalidade, seja ela privada, comercial ou acadêmica.
