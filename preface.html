Este livro é a documentação oficial do PostgreSQL. Foi escrito pelos desenvolvedores do PostgreSQL e outros voluntários em paralelo ao desenvolvimento do software do PostgreSQL. Este livro descreve todas as funcionalidades que a versão corrente do PostgreSQL oferece suporte oficialmente.

Para tornar a grande quantidade de informações sobre o PostgreSQL mais gerenciável, este livro foi organizado em várias partes. Cada parte é direcionada a uma categoria diferente de usuários, ou a usuários em diferentes estágios de sua experiência com o PostgreSQL:

  • A Parte I é uma introdução informal para novos usuários.

  • A Parte II documenta o ambiente da linguagem de consulta SQL, incluindo tipos de dados e funções, bem como ajustes de desempenho no nível de usuário. Todo usuário do PostgreSQL deve ler esta parte.

  • A Parte III descreve a instalação e administração do servidor. Todos aqueles que mantêm um servidor PostgreSQL, seja para uso privado ou para terceiros, devem ler esta parte.

  • A Parte IV descreve as interfaces de programação para aplicações cliente do PostgreSQL.

  • A Parte V contém informações para usuários avançados sobre os recursos de extensibilidade do servidor. Os tópicos incluem tipos de dados e funções definidos pelo usuário.

  • A Parte VI contém informações de referência sobre os comandos SQL, programas cliente e servidor. Esta parte auxilia as outras partes com informações estruturadas, classificadas por comando ou programa.

  • A Parte VII contém informações diversas que podem ser úteis para os desenvolvedores do PostgreSQL.
