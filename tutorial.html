Parte I. Tutorial

Seja bem-vindo ao tutorial do PostgreSQL. Este tutorial tem como objetivo apresentar o PostgreSQL, os conceitos de banco de dados relacionais e a linguagem SQL. Pressupõe-se haver algum conhecimento geral sobre como usar computadores, mas não é necessária experiência específica no sistema operacional Unix ou em programação. Este tutorial tem como objetivo proporcionar experiência prática em aspectos importantes do sistema gerenciador de banco de dados PostgreSQL. Não tem a intenção de ser um tratamento exaustivo dos tópicos abordados.

Após concluir este tutorial com sucesso, você poderá ler a Parte II para ter uma melhor compreensão da linguagem SQL, ou a Parte IV para obter informações sobre o desenvolvimento de aplicações com o PostgreSQL. Aqueles que provisionam e gerenciam sua própria instalação do PostgreSQL também devem ler a Parte III.
